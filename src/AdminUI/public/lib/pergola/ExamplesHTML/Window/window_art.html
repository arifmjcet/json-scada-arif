Window

View as standalone SVG

A pergola.Window object includes by default command buttons, menu bar, toolbar with the default set of transformation tools, and status bar. The properties hasCommands, hasToolBar, hasZoomAndPan, and hasStatus let you override this behavior. Other properties let you control geometry, position, state and appearance. The type property lets you define a mapping window ("map"), a datagrid window ("datagrid"), or a vector editing window ("editor"). A window always has scrollbars, unless you override the resizable property, and is always movable. For more solutions check the Panel examples.
