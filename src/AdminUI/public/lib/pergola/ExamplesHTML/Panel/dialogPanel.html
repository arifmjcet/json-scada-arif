Generic dialog Panel. The value of the type property defaults to "dialog". The dialog buttons are configurable. The properties "fill-opacity" and "stroke-opacity" of the pergola.Frame component can be overridden.

Panel contents can be appended manually, or by setting the contains node or function property, or by specifying a layout.
