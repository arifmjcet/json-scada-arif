Window - Load SVG Document

View as standalone SVG

Import a remote SVG document into a window or any container, with merge, reset scale and progress bar options.

In this example the imported document is encapsulated into a window. Windows have a progress bar embedded in the top bar (under the menu).

HOW

You can use the techique for deferred loading (commented code above) to update the contents (merge or replace) of a window or other specified container. Learn more in the LOADING SVG DOCUMENTS section of the Tutorial.
