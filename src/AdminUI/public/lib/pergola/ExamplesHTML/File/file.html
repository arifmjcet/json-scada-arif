File Dialog

Pergola builds the pergola.fileDialog system dialog with lazy loading.

There is no standalone example for this control. To use, invoke the pergola.getFiles() utility specifying configuration properties. A working example is in the DATAGRID example. Note that in the live example on the Pergola server, saving files is deactivated; in order to use the “Save As” voice of the File menu, you first need to put the package on your sever.
