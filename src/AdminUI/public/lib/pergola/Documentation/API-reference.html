API Reference

All Pergola things are under the pergola namespace. Pergola now uses semantic versioning. To find the current version use pergola.version.

  • Class (superclass)

Core

DOM

  • SVG() – universal SVG node builder
  • HTML() – universal HTML node builder
  • HTML_FO() – universal HTML node builder as foreighObject

node manipulation

  • removeChildren() – remove all children of specified node
  • attribute() – set a collection of attributes for specified node

class operations

  • subclass() – subclass inheritance
  • extend() – extends objects or classes
  • destroy() – deletes properties and variables

requests - load - save

  • getFiles() – retrieve the file list from specified folder on server
  • loadSVG() – loads and parses an SVG document, and appends it to the specified container
  • saveJSON() – save file (json) to specified folder
  • saveSVG() – save file (svg) to specified folder
  • xhrGET() – issue a GET request
  • xhrPOST() – issue a POST request
  • xmlToJson() – XML to JSON converter

pergola.xmlName

  • list – list of the generated xml names
  • exist() – checks if xml name to add is already used
  • add() – adds xml name returned by String.xmlNameGenerator()
  • remove() – removes assigned xml name from list (on object deletion)

pergola.Undo

  • max – levels
  • buffer – stores mementos
  • index – buffer index
  • canUndo() – buffer is not empty
  • canRedo() – buffer length is greater than index
  • add() – record an action
  • update() – shifts buffer if overflow
  • cancel() – remove latest entry in buffer
  • destroy() – checks all elements referenced (one or more reference) in the buffer objects and deletes them where applicable
  • flush() – reinitialize buffer
  • undo() – handler; if canUndo, invokes objects' undo callback
  • redo() – handler; if canRedo, invokes objects' redo callback
  • handleEvent() – event handler; forks to sub-handlers undo / redo
  • register() – activate recording of this action (objects to be monitored must instantiate the class first)
  • disable() – stop monitoring. Call this on action termination

pergola.Timer

  • initialize() – start a timer process with specified frequence (if any) and delay (if any)
  • setInterval() – called internally if frequence was specified
  • clear() – clear interval and/or timeout; optionally destroy instance

pergola.Key

Prototype methods

  • register() – enables keyboard monitoring; starts process
  • release() – stops keyboard monitoring; ends current process optionally destroy instance

Class properties and methods

  • active – reference to the current typing process
  • keydown() – default generic keydown handler; handles keyCode (8, 9, 13, 46, 112–123)
  • keypress() – default keypress handler; handles charCode; alphanumeric
  • keyup() – default keyup handler; no action
  • keypressNum() – optional keypress handler; handles charCode; filters numeric values
  • keypressInt() – optional keypress handler; handles charCode; filters integer values
  • keypressPos() – optional keypress handler; handles charCode; filters positive numeric values
  • keypressPosInt() – optional keypress handler; handles charCode; filters positive integer values
  • keypressHex() – optional keypress handler; handles charCode; filters hexadecimal values

pergola.geom

  • intersectionPoint() – get intersection point between two lines
  • intersect() – test intersection
  • pointBetween() – test if point is between two points
  • pointOnLine() – test if point is on a segment
  • getPDistance() – get shortest distance from point to segment
  • getPPoint() – get perpendicular point on line from point
  • getP2Pdistance() – get distance between two points
  • centroid() – get true centroid
  • crossProduct() – cross product
  • angle
    • angle.degrees() – get the rotation angle of a line in degrees
    • angle.radians() – get the rotation angle of a line in radians
    • angle.toDegrees() – convert radians (number or string) to degrees
    • angle.toRadians() – convert degrees (number or string) to radians
    • angle.angleBetween2Lines() – get angle between two lines
    • angle.constrain() – constrain line to fixed angles from specified number of cardinals
  • area
    • area.triangle() – computes triangle area based on sides
    • area.polygon() – computes polygon area based on vertices (clockwise or counterclockwise)

pergola.color

Properties

  • rgbTable – table of color values
  • neutral – the current theme color is neutral
  • components – object with r, g, b properties
  • ingredients – array of sorted color components of the current theme color
  • themeLuminosity – the luminosity (0–255) of the current theme color
  • keywordNames – object with the color keywords name/value pairs.
  • customColorNames – object defining custom color keywords

Methods

  • convert() – converts a color string in any legal format (including color keyword and custom color keyword) to the specified format
  • hslToRgb() – converts hsl to rgb
  • hexToRGB() – converts hexadecimal (complete string with #) to rgb
  • rgbToHex() – converts single rgb value (number) to hexadecimal (always returns two digits from 00 to FF)
  • lab() – used by system
  • shade() – get a shade of current theme color from specified array of ingredients in %, and positive or negative brightness factor
  • rgbTablePopulate
    • rgbTablePopulate.basic() – populates a 65536 rgb colors table
    • rgbTablePopulate.extended() – populates a 393216 rgb colors table

pergola.geo

  • distance() – Computes the distance between two geo-locations
  • kmToMi() – convert kilometers to miles
  • kmToNmi() – convert kilometers to nautical miles
  • formatMiles() – formats to miles-feet-yards-inches depending on magnitude
  • formatKm() – formats to kilometers-meters-centimeters depending on magnitude

pergola.contextmenuManager

  • browser() – handler for "contextmenu" event registered on the window object
  • custom() – handler for "mousedown" event registered on the window object
  • activate() – activates custom contextmenu

pergola.debugger

  • toggleOn() – opens the debugger panel
  • toggleOff() – closes the debugger panel
  • vars() – shows property/value pairs in parameter object

pergola.lazyLoad

  • fileDialog() – Creates the pergola.fileDialog panel
  • findDialog() – Creates the pergola.findDialog panel
  • colorpickerDialog() – Creates the pergola.colorpicker panel

miscellaneous

  • decimalToHex() – converter
  • find() – utility function to initialize and open the pergola.findDialog
  • getColorpicker() – utility function to initialize and open the pergola.colorpicker
  • handleContextmenu() – system context menu management
  • handleCustomContextmenu() – custom context menu management
  • height() – get the computedStyle.height of HTML container, or of root document in standalone SVG
  • hexToDecimal() – converter
  • message() – pointer coordinates in HTML + SVG context, or standalone SVG
  • mousePoint() – pointer coordinates in HTML + SVG context, or standalone SVG
  • resize() – redraws the interface on resize event
  • setCursor() – set SVG "cursor" attribute (workaround)
  • symbol() – returns a group containing a symbol (from library or defined on the fly). Use for symbols or icons
  • toArray() – store property values into array (avoids conflict with other extension libraries)
  • use() – use a preset shape from the shapes library with SVG attributes specified in the object passed
  • width() – get the computedStyle.width of HTML container, or of root document in standalone SVG

system constants

  • version – the current version
  • browser – browser identifiers
  • locale – browser's language
  • debug – use debugger (as specified in config file)
  • decimals – default decimal places (as specified in config file)
  • getDir – default file on server for getting directory listings
  • NF – Intl.NumberFormat implemented
  • DTF – Intl.DateTimeFormat implemented
  • mutationEvnt – MutationEvents 2 implemented
  • mutationObserver – mutationObserver implemented
  • ns – namespace prefixes
  • path – path to pergola folder (as specified in config file)
  • radixChar – table of locale decimal delimiters
  • remote – lets pergola know if it's running in remote or local mode
  • systemMenu – determines whether pergola has a system menu on the taskbar (as specified in config file)
  • taskbarLogo – lets you place a logo on the taskbar (as specified in config file)
  • theme – the current theme color (as specified in config file)
  • undos – number of undos (as specified in config file)
  • writeFile – default file on server for writing files

environment variables

  • pergola.removeSelection – removal of current selection range
  • pergola.preventMousemoveDefault – affects selection
  • pergola.preventMousedownDefault – affects selection
  • pergola.preemptiveIsUp – prevents any action on the interface if a preemptive dialog is up
  • pergola.contextmenuEnabled – allows to switch to a custom context menu

  • doc – document.documentElement in SVG mode, or outmost <svg> element in HTML mode
  • preemptive – the <g> element containing preemptive panels
  • systemComponents – the <g> element containing system components
  • user – the <g> element default container for user objects

shortcuts

The shortcuts are not in the pergola namespace, they are global variables.
The $ symbol is not used in the conventional way. In Pergola all elements and component elements that are likely
to be manipulated, or that can be manipulated are referenced, and you never need to invoke getElementById().

  • $C() – alias for pergola.SVG()
  • $H() – alias for pergola.HTML()
  • $M() – alias for pergola.message()
  • $D() – alias for pergola.debugger.vars()

  • checkCallback() – checks the type (function or string) of the callback

  • sweep() – short for deleting specified properties
  • getFiles() – Reference to the pergola.getFiles() utility function
  • getColorpicker() – Reference to the pergola.getFiles() utility function

  • measureByRulerTool() – measures distance (maps)
  • measureItinerary() – calculates itinerary (maps)
  • zoomTimerInit() – initializes zoom timer
  • mapZoomFadeout() – continues zoom to next or previous integer value
  • mapZoomDownByButton() – applies map zoom
  • mapZoomUpByButton() – applies map zoom
  • zoomDownByButton() – applies zoom
  • zoomUpByButton() – applies map zoom
  • quickTipFormat() – checks format and origin (library or user) of quic ktip
  • quickTipInit() – initializes quicktip timer
  • rollover() – button hover effects
  • videoInverse() – button hover text effect
  • buttonSelect() – applies effect on selected button
  • buttonDeselect() – removes effect from deselected button
  • textCenterVertical() – center text vertically
  • textShow() – show text; for objects that have a text property
  • textHide() – hide text; for objects that have a text property
  • textDataUpdate() – update text value; for objects that have a text property

Built-in objects extensions

Number

  • evenToOdd() – get the odd smaller than this, where this is even
  • oddToEven() – get the even smaller than this, where this is odd
  • isEven() – test if this is even
  • isInteger() – test if this is an integer
  • trim() – get this number (as Number) with the specified number of decimal places
  • toRad() – convert degrees to radians
  • toDeg() – convert radians to degrees

String

  • xmlNameStartChar – regular expression; illegal XML name start characters
  • xmlNameChar – regular expression; illegal XML name characters
  • toXML() – make a string XML conformant
  • xmlNameGenerator() – get a unique XML string composed of "XML_" + 32-bit hex number
  • unformatNumber() – convert a locale formatted number to JavaScript number
  • width() – get this bbox width (rounded) with the specified font parameters (size, family, weight, letter-spacing, kerning)
  • height() – get this bbox height (rounded) with the specified font parameters (size, family)
  • truncate() – get this truncated by the specified number of characters
  • truncateWithDots() – get this truncated to the specified width, with the specified font parameters (size, family, weight, letter-spacing, kerning), + "..."
  • centerVertical() – get the y value (for text element) of this, from specified height and font parameters (size, family)
  • brighten() – get this (color string in any legal format) brightened by the specified factor
  • darken() – get this (color string in any legal format) darkened by the specified factor
