pergola.UpDownButton

System events: YES.

User events and functions: NO.

This class is specifically designed for the pergola.Input class; if the hasButtons is set (initial true) for the Input instance, the input box is a spinner. Subclass of pergola.Button; inherits pergola.Button.prototype.
