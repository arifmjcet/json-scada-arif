Quick Tips Library

pergola.quickTip (defined in lib/qtips.js) is an object defining object properties. Each object property must define the tip string or array of strings property. See the QuickTip class for other properties that can be specified to override the default behavior of a quick tip. Read also the Quick Tips library chapter in the tutorial.
