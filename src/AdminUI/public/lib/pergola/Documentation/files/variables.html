Environment Variables

The following properties are called variables to differentiate them from the system constants properties, which should never be reassigned. Except for contextmenuEnabled the state of these properties is handled by the system, and you don't normally need to specifically set them.

pergola.contextmenuEnabled

Boolean; initial true. If false it allows to switch to a custom context menu, or to simply disable the browser's context menu. Learn more in the CUSTOM CONTEXT MENU section of the Tutorial.

pergola.preemptiveIsUp

Object reference; initial null. Prevents any action on the interface if a preemptive dialog is open.

pergola.preventMousemoveDefault

Boolean; initial true. Prevents selection on mousemove event.

pergola.preventMousedownDefault

Boolean; initial false. Prevents selection on mousedown event.

pergola.removeSelection

Boolean; initial false. If true it cancels selection on mousemove event.
