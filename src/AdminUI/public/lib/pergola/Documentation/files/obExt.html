Built-in objects extensions

The Number and String prototype extensions included in Pergola is a small collection of methods that are used by the system, and therefore must not be replaced. If you need comprehensive Number and/or String libraries for your application, they should preferably be loaded before the built-inProtoExt.js file, to ensure that none of the plugged-in library's methods overrides any of the extension methods defined by Pergola.
