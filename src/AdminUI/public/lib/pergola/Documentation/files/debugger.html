Debugger

Use the pergola.debugger to show variables and properties name/value pairs in the debugger panel. Activate the debugger in the config.js file bound to each project. Learn more in the USING THE DEBUGGER section of the Tutorial.

pergola.debugger

toggleOn()

Expands the debugger panel (click on debugger symbol in top-left corner).

toggleOff()

Collapses the debugger panel (click on close symbol in top-left corner).

vars(input)

Shortcut: $D. input is an object with arbitrary property/value pairs (normally the same). The property name uses the string notation. Use example: 

SD({"myRect.x" : myRect.x});
The debugger panel displays: myRect.x = value of myRect.x
