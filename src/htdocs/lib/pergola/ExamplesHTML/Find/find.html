Find Dialog

View as standalone SVG

Pergola builds the pergola.findDialog system dialog with lazy loading. It is initialized by invoking the pergola.find() utility function, passing an object parameter with the user object and callback function (or method name string) properties. Note that in the context created by instances of Pergola classes inheriting from the Class superclass, you can state this.find(). A selection range is created for the current occurence. The dialog is preemptive.

See also the datagrid example (finds occurrences within string, number, date, and boolean data types).
