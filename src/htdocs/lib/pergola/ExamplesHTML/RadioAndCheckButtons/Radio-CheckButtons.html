Radio Button and Check-box

View as standalone SVG

For a group of sibling RadioButton objects you can define the manager object property. User functions can be assigned to the radio buttons and/or to the owner object.

For a group of sibling CheckBox objects you can define the manager object property. The checked array property is dynamically added to manage the state of the check-boxes. User functions can be assigned to the check-boxes and/or to the owner object.
