Legend

View as standalone SVG

The new pergola.Legend lets you define one or more SVG primitives or images for the legend lines.

The caption can have more than one line, and you can also specify text attributes using SVG vocabulary and grammar. The Vertical spacing and text offset can be overridden.

The class has toggleOn(), toggleOff() and move() prototype methods.
