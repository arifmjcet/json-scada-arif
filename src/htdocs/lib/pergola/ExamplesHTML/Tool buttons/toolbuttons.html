Tool Button – Command Button

View as standalone SVG

pergola.ToolButton and pergola.CommandButton are subclasses of pergola.Button. A tool button with the exclusive property set acts as a radio button. Other properties like selected, suspended, type, and cursor are designed for tools management.

The pergola.Window class has the addTools() prototype method for adding tools with their predefined functionalities. A window implements full management of its tools.
