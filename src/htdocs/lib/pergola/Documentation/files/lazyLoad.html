Lazy Loading

Object defining functions that create, on first use, system components that are not created at runtime.

pergola.lazyLoad

fileDialog()

Creates the pergola.fileDialog, which is used for file Open and Save As operations. See pergola.getFiles() to learn how to initialize the dialog.

findDialog()

Creates the pergola.findDialog. See pergola.find() to learn how to initialize the dialog.

colorpickerDialog()

Creates the pergola.colorpicker.
