pergola.Preview

System events: YES.

User events and functions: NO.

System component. This class is a dependency of pergola.Window and shows an updated preview (clone of the window's doc.transformable layer, reflecting its current rendering) when hovering a pergola.WindowTab in tabsDock. You don't need to instantiate this class, or invoke any of its methods explicitly.

METHODS

show()

Shows the preview. You don't normally need to invoke this method explicitly.

hide()

Hides the preview. You don't normally need to invoke this method explicitly.

update()

Updates the preview of this owner window on transformations and DOM changes. Dynamic updating is disabled for windows of type "datagrid", where only the first shot of visible data is taken at runtime. You don't normally need to invoke this method explicitly.
