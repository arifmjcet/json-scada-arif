pergola.PopupListItem

System events: YES.

User events and functions: NO (callback funtion is defined for pergola.Selector object).

System component. Subclass of pergola.ListItem; inherits prototype of pergola.ListItem. This class is specifically designed as component for the pergola.Selector class. You don't need to instantiate this class, or invoke any of its methods explicitly.
