pergola.Suffix

System events: YES.

User events and functions: NO.

Use this component to add a suffix (units, %, currency...) to selectors, input boxes (if you're not using a formatting method for the value), etc. Subclass of pergola.Caption; inherits pergola.Caption's prototype.

Listed: class properties/methods; prototype properties and methods; instance properties. Prototype properties and methods inherited by the Class superclass are not listed.

class

None

prototype

PROPERTIES

parent

pergola.user. For this class this property should normally be overridden to designate the container of the instantiating object.

position

"right" – string (values: "left"; "right"). For documents with right-to-left text direction edit the skin file (pergola.presentationAttributes.suffix) rather than overriding this property on a case by case basis.

"text-anchor"

"start" – string. For documents with right-to-left text direction edit the skin file (pergola.presentationAttributes.suffix) rather than overriding this property on a case by case basis.

activator

false – boolean. If true, the caption is sensitive to events and can activate the control to which it relates (selectors, input boxes).
